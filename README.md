# Installation
```bash
git clone git@github.com:insoul/gitup.git ~/.gitup
echo 'export PATH=~/.gitup:$PATH' >> ~/.bash_profile
```
# Configuration
```bash
cp ~/.gitup/config.yml.example ~/.gitup/config.yml`
```
Edit `~/.gitup/config.yml`

# Usage
```bash
gitup send

gitup send --server dev_server --restart
```
